# Screenshot Knowledge Manager

A local-first, privacy-focused productivity tool that transforms raw screenshots into a searchable, categorized knowledge base using automated OCR and LLM-assisted indexing.

## Features
* **Automated Watchdog:** Real-time processing of new files in your system.
* **Privacy Shield:** Automated filtering of sensitive information like OTPs and API keys.
* **Search Engine:** Built on SQLite FTS5 for lightning-fast retrieval of your digital notes.
* **Performance:** Optimized for sub-1s OCR latency and minimal memory footprint.

## Stress Test
Successfully validated with a 300-file repository while maintaining index consistency and query speed.

## Usage
Simply point the script at your screenshots directory, and it will handle the organization and indexing automatically.
